Linguagens de programação ensinadas na Dev School

Python

Linguagem versátil usada para análise de dados, desenvolvimento web, automoção, big data e muito mais!

JavaScript

Linguagem usada no desenvolvimento web e mobile. Antes restrita ao client side, hoje é amplamente usada no server side.

Linguagem voltada para a parte Back-end do desenvolvimento web. Está há muito tempo no mercado e foi a linguagem usada na criação do Facebook.

Java

Linguagem extremamente versátil. Pode ser usada para programar tudo, desde relógios e televisões até desenvolvimento mobile e de games.
